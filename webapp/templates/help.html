Boardgames API Documentation

This API provides access to boardgame data from our database. All responses are in JSON format.

Base URL

All API endpoints are relative to: /api

Endpoints

1. Get All Games

GET /games

Returns a list of all boardgames in the database.

Example Response:

{
  "games": [
    {
      "id": 0,
      "name": "Terraforming Mars: Prelude"
    },
    {
      "id": 1,
      "name": "Brass: Birmingham"
    },
    ...
  ]
}

2. Get Game by ID

GET /games/<id>

Returns detailed information about a specific game by its ID.

Parameters:

Name Type Description
id Integer The unique identifier of the game

Example Request:

/api/games/8 (Terraforming Mars)

Example Response:

{
  "game": {
    "id": 8,
    "name": "Terraforming Mars",
    "designer": "Jacob Fryxelius",
    "publisher": "FryxGames",
    "yearPublished": 2016,
    "minPlayers": 1,
    "maxPlayers": 5,
    "playingTime": 120,
    "minPlaytime": 120,
    "maxPlaytime": 120,
    "age": 12,
    "average": 8.43,
    "rank": 3,
    "description": "Compete to transform Mars into a habitable planet by..."
  }
}

3. Get Game by Name

Returns game(s) that match the provided name (partial match supported).

Parameters:

Name Type URL Description
name String /games/name/<name> Full or partial name of the game
artist String /games/artist/<artist> Name of person who created artwork for a game
desginer String /games/designer/<designer> Name of person who created the design of a game
maxplayers String /games/maxplayers/<maxplayers> The maximum amount of people a game can support
minplayers String /games/minplayers/<minplayers> The minimum amount of people a game can support
minplaytime String /games/minplaytime/<minplaytime> What is the least amount of time a game takes to play

Example Request (For entering a the name parameter):

/api/games/name/catan

Example Response:

{
  "games": [
    {
      "id": 701,
      "name": "CATAN"
    },
    {
      "id": 706,
      "name": "CATAN: Seafarers" 
    },
    {
      "id": 399,
      "name": "CATAN: Cities & Knights"
    }
  ]
